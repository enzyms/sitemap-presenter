Welcome to the Test Page

This page demonstrates the feedback widget functionality. Click the orange "Feedback" button in the bottom-right corner to enter placement mode, then click anywhere on the page to leave a feedback marker.

Each marker can have multiple comments, and you can resolve or reopen feedback as needed. The widget syncs in realtime via Supabase.

1
Easy Installation

Just add a single script tag to your page. No complex configuration required.

2
Visual Feedback

Click anywhere on the page to leave a marker. Great for design reviews and bug reports.

3
Realtime Sync

All feedback syncs instantly across all viewers. Powered by Supabase Realtime.

4
Element Anchoring

Markers anchor to DOM elements, so they stay in place even when the page changes.
